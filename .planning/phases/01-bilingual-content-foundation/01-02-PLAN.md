**As a** Besucher (Recruiter, Tech-Lead oder Reviewer), **I want to** sämtliche Portfolio-Inhalte vollständig auf Deutsch und Englisch unter `/de` und `/en` einer deployten, performance-überwachten Site lesen und die Sprache umschalten, **so that** ich Lasses Werdegang, Projekte und Kontaktmöglichkeiten in meiner Sprache beurteilen kann.

## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Repo → Vercel production (lsiem.de) | A wrong deploy replaces the live site with an unfinished skeleton (D-08) |
| Repo → GitHub Actions | Third-party actions execute with repo access |
| Public repo / CI logs | Confidential terms must never appear (D-03) |
| Visitor → Vercel Analytics | Personal data handling is delegated to the platform |
