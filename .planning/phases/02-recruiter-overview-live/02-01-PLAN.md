<objective>
Complete the functional recruiter overview so a hurried visitor finds every fact fast, on the existing single-scroll `/[locale]` page.

**Phase goal (user story):** **As a** hurried recruiter or tech-lead, **I want to** find who Lasse is, what he does, his timeline, skills, and contact/CV on a fast bilingual page, **so that** I can evaluate him in under 30 seconds.

This plan is the anchor vertical slice: it takes the interim overview (which already renders hero/career/projects/skills/contact) and finishes it into a complete, accessible, one-click-contactable recruiter layer. Purpose: after this plan the site is a complete, deployable, accessible recruiter overview.

</objective>

| visitor browser → static site | Read-only SSG page; no forms, no state-changing actions, no auth |

| Threat ID | Category | Component | Severity | Disposition | Mitigation Plan |

| T-02-06 | Tampering (XSS) | authored content rendered as text | low | mitigate | All content rendered as React text children (auto-escaped); no `dangerouslySetInnerHTML` introduced in this plan |
